PeluApp

PeluApp es una aplicación móvil diseñada para facilitar la gestión de citas y servicios en una peluquería, permitiendo a los clientes reservar citas, explorar servicios y consultar horarios disponibles de manera sencilla, mientras que los empleados pueden gestionar sus citas de forma eficiente. La aplicación está desarrollada en Java utilizando SQLite como base de datos y Android Studio como plataforma de desarrollo.
